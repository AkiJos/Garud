# Garud
Garud searches for hashes online and it currently supports Hybrid Analysis, VirusTotal, Cape Sandbox

usage: Garud.py [-h] [--ha_key HA_KEY] [--vt_key VT_KEY] --hash HASH
                [--hash_val HASH_VAL]
